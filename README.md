## 🛠️ Installation & Setup

1. **Clone the repo**

```bash
git clone https://github.com/Tameem-Elmasry/tameem_portfolio.git
cd tameem_portfolio
```

2. **Install dependencies**

```bash
npm install
```

3. **Run the development server**

```bash
npm run dev
```

4. **Build for production**

```bash
npm run build
```
